# Pyro-Maker-Show
Проект пульта для запуска пиротехники на Arduino и C#

# Назначение файлов

* Memo - Шаблон для памятки Photoshoop
* PyroCTRL - Скетч для Arduino IDE
* Pyro Maker Show - Готовая программа редактора
* Script Editor - Исходник редактора на C#
